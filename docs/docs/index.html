QTM 350 - Data Science Computing

Course Description

This course will introduce students to key tools in the modern data science workflow, focusing on three Rs that are essential for success in data science: reliability, reproducibility, and robustness. The course will cover the following topics: 1) command line interfaces and vim; 2) version control with Git and GitHub; 3) literate programming and reproducible research with Quarto and Jupyter Notebooks; 4) data storage and manipulation with SQL and Pandas; 5) data visualisation with Matplotlib, Seaborn and plotnine; 6) parallel computing with Dask; 7) GitHub Co-Pilot and AI-assisted programming; 8) Docker and containerisation.

Learning Outcomes

By the end of this course, students will be able to:

  • Use data science tools for project collaboration and version control
  • Apply advanced techniques for data storage, manipulation, and querying
  • Create clear data visualisations and write well-documented code
  • Use AI tools to help with programming tasks
  • Understand the basics of containerisation and parallel computing

Course Requirements

  • Prerequisites: Some knowledge of Python or R is recommended, and familiarity with basic data manipulation and visualisation techniques is helpful. No prior experience with the tools covered in the course is required.
  • Software: Anaconda distribution of Python 3.x, VS Code, PostgreSQL, GitHub Desktop, Git, Docker, Quarto, Dask, GitHub Co-Pilot.

Grading

  • Assignments: 50%
  • Class Quizzes: 30%
  • Final Project: 20%
